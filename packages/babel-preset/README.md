# @patternplate/babel-preset

> @babel preset for compilation of @patternplate projects


## Getting started

```
npm install --save-dev @patternplate/babel-preset
```

## Usage

```json
{
  "name": "package",
  "babel": {
    "presets": [
      [
        "@patternplate/babel-preset",
        {
          "targets": ["web", "node"]
        }
      ]
    ]
  } 
}

```
